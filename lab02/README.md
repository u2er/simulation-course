# Отчет: Численное решение уравнения теплопроводности

## 1. Постановка задачи
Моделирование распространения тепла в одномерной пластине методом конечных разностей.
*   **Метод решения:** Метод сеток
*   **Реализация:** Вычислительное ядро на **C**, интерфейс и визуализация на **Python**.

## 2. Параметры эксперимента
Для генерации итоговой таблицы использовались следующие физические и геометрические параметры:
*   **Материал:** Уран ($\rho=19050$, $c=116$, $\lambda=27.5$).
*   **Длина пластины ($L$):** 0.2 м.
*   **Граничные условия:** $T_{left} = 100^\circ C$, $T_{right} = 20^\circ C$.
*   **Начальные условия:** $T_{init} = 20^\circ C$.
*   **Модельное время ($t$):** 120.0 с.

## 3. Результаты моделирования
Таблица значений температуры ($^\circ C$) в центре пластины в зависимости от шага по пространству ($h$) и времени ($\tau$).

| $h$ (м) \ $\tau$ (с) | 0.1 | 0.01 | 0.001 | 0.0001 |
| :--- | :--- | :--- | :--- | :--- |
| **0.1** | 30.32672 | 30.32771 | 30.32781 | 30.32782 |
| **0.01** | 25.45501 | 25.45429 | 25.45421 | 25.45421 |
| **0.001** | 25.38385 | 25.38314 | 25.38307 | 25.38306 |
| **0.0001** | 25.38314 | 25.38243 | 25.38235 | 25.38235 |

## 4. Анализ и Выводы

1.  **Сходимость по пространству ($h$):**
    *   При грубом шаге $h=0.1$ м наблюдается значительная погрешность (температура $\approx 30.3^\circ C$).
    *   При уменьшении шага до $h=0.01$ м результат резко уточняется ($\approx 25.45^\circ C$).
    *   Дальнейшее уменьшение до $h=0.001$ м и $h=0.0001$ м дает сходимость к значению $\approx 25.38^\circ C$. Разница между последними двумя шагами минимальна, что говорит о достижении достаточной точности аппроксимации.

2.  **Сходимость по времени ($\tau$):**
    *   Влияние шага по времени на результат в данной задаче значительно меньше, чем влияние пространственного шага.
    *   Разница между $\tau=0.1$ и $\tau=0.0001$ (при фиксированном $h=0.001$) составляет всего $\approx 0.0008^\circ C$.


**Итог:** Для данной задачи оптимальными параметрами с точки зрения соотношения "точность/ресурсы" являются $h=0.001$ м и $\tau=0.01 \dots 0.1$ с.
